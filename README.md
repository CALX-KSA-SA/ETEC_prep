# CALX ETEC School Accreditation Website

A modern, high-performance, bilingual (Arabic/English) website designed to help international schools in Saudi Arabia achieve ETEC (Education and Training Evaluation Commission) accreditation.

    - Dynamic compliance dashboard.
    - Mobile-optimized interface.
- **PADET Methodology**: Interactive visualization of the Planning, Assessment, Development, Execution, and Tracking methodology.
- **High Performance**: zero-dependency architecture using vanilla JavaScript and Tailwind CSS (CDN), optimized with lazy loading and deferred scripts for fast initial paint.
- **Responsive Design**: Mobile-first approach ensuring a seamless experience across all devices.
